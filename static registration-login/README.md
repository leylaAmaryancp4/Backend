Static Registration/Login with Express
Project Overview

This project is a Node.js + Express web application that implements a classic registration and login system with:

Two static HTML pages: register.html and login.html

CSS styling for each page

Backend API to handle registration and login

Optional image upload for user profiles

Server-side validation for all inputs

JSON file storage for user data

It is designed to demonstrate basic backend functionality including routing, static file hosting, form handling, multipart file uploads, and validation.

 Installation

Clone the repository

git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>


 Install dependencies

npm install


Start the server

For development (auto-restart on changes):

npm run dev

For production:

npm start


Open in browser

http://localhost:3000/register.html
